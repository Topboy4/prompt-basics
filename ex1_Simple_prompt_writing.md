Simple Prompt Writing

ANSWER: Python, Java and Node.js
Write a question style prompt (e.g., “What are the benefits of Python for AI?”).

ANSWER: the of python for AI are;

Compare how the model answers each type.

ANSWER: The direct instruction prompt resulted in a concise and structured output, mainly in list format. The question-style prompt generated a more descriptive and explanatory answer.

The difference shows that:

Instruction prompts produce more controlled and formatted outputs.

Question prompts encourage broader explanations and additional details.
